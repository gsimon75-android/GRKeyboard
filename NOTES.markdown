
Arabic gestures
===============

Consonants
----------
Base character          tap
Alternate char.         down
Base + shaddah          up
Alt. + shaddah          down + up

Vowels marks (after a consonant)
--------------------------------
damma/kasrah/fatha      tap
double form             down

Y-W-A
-----
Base character          tap
Base + maddah above     down
Y/W + shaddah           up
Alif waslah             up
Base + hamzah above     down + up

--------

#V
